HTML CSS Table Cheatsheet

HTML

HTML Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row data.
<tfoot> Table Foot The set of rows defining the footer in a table.

CSS

Item Details Description
1. .styled-table tbody tr:nth-of-type(even) For row with no calss
2. .styled-table tbody tr:last-of-type For the last row
3. .styled-table tbody tr.active-row For the role with class="active-row"
